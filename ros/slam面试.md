# 1	数学与理论基础
### 1.1	什么是李群、李代数？

李群是具有群结构的微分光滑流行，常见的主要是三维旋转矩阵群和三维变换矩阵群，也就是$SO(3)$和$SE(3)$,李代数是李群在单位元空间附近的一个切空间，每个李群都有对应的李代数

### 1.2 李群和李代数的关系，为什么需要他们？

* 李群和李代数可以通过对数和指数映射进行相互转换，使用对数运算将李群转换为李代数，使用指数运算将李代数转换为李群。
* 由于旋转矩阵对于加法运算不是封闭的，只对乘法运算封闭，导致无法使用标准的微积分($R+\Delta R$)对其旋转矩阵$R$进行求导等运算，因此需要引入对加法和乘法都封闭的李代数。利用李代数，可以通过给位姿加左扰动，从而进行求导


### 1.3 什么是KF，EKF，ESKF
**KF(卡尔曼滤波)**

* **预测步 (Predict)：**
* 状态预测：$\hat{x}_k^- = A \hat{x}_{k-1} + B u_k$
* 协方差预测：$P_k^- = A P_{k-1} A^T + Q$

* **更新步 (Update)：**
* 计算卡尔曼增益：$K_k = P_k^- C^T (C P_k^- C^T + R)^{-1}$
* 状态更新（修正）：$\hat{x}_k = \hat{x}_k^- + K_k (z_k - C \hat{x}_k^-)$
* 协方差更新：$P_k = (I - K_k C) P_k^-$

**EKF (Extended Kalman Filter)**

* **预测步：**
* 状态预测：通过非线性函数直接积分 $\hat{x}_k^- = f(\hat{x}_{k-1}, u_k)$。
* 协方差预测：利用 **Jacobian 矩阵** $F$ 将非线性函数在当前点一阶展开：$P_k^- = F P_{k-1} F^T + Q$。

* **更新步：**
* 计算卡尔曼增益：同样利用观测函数的 Jacobian 矩阵 $H$。
* 状态更新：使用非线性观测残差 $(z_k - h(\hat{x}_k^-))$。

**ESKF (Error-State Kalman Filter)**
* **流程：**
1. **预测标称状态：** 不考虑噪声，直接进行 IMU 动力学积分（例如更新旋转、位置）。
2. **预测误差状态协方差：** 误差状态通常是线性的（如小角度假设），预测其协方差 $P$。
3. **观测更新：** 当观测值（如 Lidar 配准后的位姿）到来，计算**误差状态**的后验估计 $\delta \hat{x}$。
4. **状态注入 (Injection)：** 将估计出的误差 $\delta \hat{x}$ 补偿回标称状态（例如通过李群乘法更新旋转）。
5. **重置 (Reset)：** 将误差状态清零，更新协方差 $P$。

### 1.4 KF中的$Q$和$R$分别是什么，怎么调节？
$Q$是过程噪声的协方差矩阵，$R$是观测过程噪声的协方差矩阵。过程噪声主要是描述系统模型的不确定性，如果系统模型的精度比较高，比如针对高精度的IMU的积分模型时，可以把$Q$的值调小，否则，调大。观测过程噪声主要根据传感器精度进行调整，如使用的是高精度的激光雷达电云进行配准，可以适当调小，否则，调大。在实际使用时，也可以先保守初始化后再调节。

### 1.5 ESKF中的$bias$的reset方法有哪几种，有什么区别？
1. 标准重置，将最优误差状态加到名义零偏上（$b+ \delta b$）,然后将$ \delta b$清零，协方差矩阵保持不变
2. 阻尼重置，引入阻尼系数$k$，$b=b+k\delta b$,\delta b$清零

### 1.6 传感器标定有什么，以及传感器的时间戳同步应该怎么做呢？
1. 内参和外参标定，内参如相机使用棋盘格标定焦距，光心位置和畸变系数，IMU内参标定零偏和白噪，LIDAR内参标定较少。外参标定主要是标定LIDAR、IMU、相机之间的位置关系。
2. 传感器时间戳同步分为硬件，软件，算法同步。硬件同步，当有GPS时，由MCU来捕获gps模块发送的UTC时间，MCU以固定的频率向相机发送PWM曝光脉冲，触发传感器。软件同步，由操作系统的cpu来打时间戳。算法同步，把传感器之间的时间差作为一个状态量放入状态向量中一起估计

### 1.7 如何理解IMU的噪声？请问IMU的测量噪声和零偏各自属于什么数学模型呢？在数学表征中各自又该如何合理构建数学表达呢
 IMU的噪声主要是白噪和零偏，其中白噪是测量时由电子元器件导致的不可避免的噪声，长期均值为0，符合高斯白噪模型。零偏是imu在长期静止时，其测量均值偏移了基准0，导致白噪也会在其附近跳动，这个零偏是可以标定的值，属于随机游走模型。

 ### 1.8 简述下几种常见LiDAR传感器的数据获取原理？机械式和固态式LiDAR的优缺点你是怎么理解的呢？
**1. 测距原理**
 * TOF(飞行时间法):发射一束极短的激光脉冲，当遇到障碍物反射回来时计算运动时间，乘以光速可以得到距离。
 * FMCW法：发射频率随时间线性变化的激光，接受到反射回激光时，通过与本地的固有频率进行混频计算距离。且可以得到速度。
 * 相位法：发射连续的幅度调制激光，通过对比发射波和接收波的相位差来计算距离

 **2. 机械、半固态、固态**
 * 机械：将激光发射器和接收器阵列垂直排列，由底部马达带动整个结构旋转得到结果。优点是视角广，扫描线束均匀，缺点是价格昂贵，且寿命与可靠性差。
 * 半固态：收发模块是固定不动的，由微振镜的高频振动向不同方向反射光线，或者由转动的多边形棱角反射将其反射到不同角度。优点是体积小，物理结构较简单，缺点是视角有限。
 * 纯固态：泛光式或者光学相控阵。

### 1.9 GNSS或者RTK的数据有什么特点？
标准GNSS的精度大约是1m-10m的级别，rtk采用载波相位和基站差分数据，而RTK的精度可以达到1cm-5cm，二者的数据不会随时间产生累积误差，每次测量都是独立的，但是容易收到环境影响，在高楼，森林等地区的精度可能很大。

### 1.10 你如何理解图优化方法和滤波方法？请简述两者的优缺点？
1. 滤波方法基于马尔科夫假设，强调当前时刻系统的状态仅由上一时刻决定，滤波器会一边处理新的数据，一边丢弃旧的数据，带来了高效的计算效率和极低的延迟，但是由于依赖于历史状态，一旦过程中出现错误，就无法消除。
2. 图优化方法多基于非线性最小二乘法，保留了历史状态，可以在优化过程中多次迭代，达到极高的精度，但是计算了庞大，实时性一般

# 2 点云配准

### 2.1 点云配准中的三大假设是什么？
1. 初始位姿假设，点云配准不是从0开始的，而是假设已经有一个初始的估计变换R，t，可以是由IMU得到的初始位姿估计
2. 对应关系假设，每个需要配准的特征点云都应该在源点云能找到对应的点进行配准
3. 环境静态与刚体假设，假设环境是静止不动的，且点云代表的物体是刚体

### 2.2 点云配准的流程
1. 点云下采样，使用去噪，滤波等工具减少点的数量，提升计算速度。
2. 特征提取，提取角点，平面点或者SIFT特征点
3. 特征匹配，使用kd-tree找到原点云附近的对应点
4. 误匹配剔除，剔除由于重复纹理等产生的误匹配
5. 粗配准，计算一个初始的粗略位姿
6. 精配准，使用icp，ndt等方法迭代配准点云

### 2.3 ICP, GICP, NDT的工作流程
**1. ICP（Iterative Closest Point）**  
 包含point-to-point和point-to-plane，运行时会为每一个点寻找在目标点云中的对应点（距离最近点），对应平面（距离最近的三个点构成的平面），通过匹配得到位姿R，t。
 * 优点：原理简单，易于实现，初始位姿良好时精度较高
 * 缺点： 高度依赖初始值，点云一开始比较远的时候容易陷入局部最优解。

 **2. GICP（Generalized ICP）**
认为每一个点都服从高斯分布，计算每个点附近的邻域协方差矩阵（局部表面形状），优化的是源点云与目标点云高斯分布之间的马氏距离，体现为一种面到面的配准
* 优点: 鲁棒性强，结合局部曲面特征使得精度配准精度更高
* 缺点：同样依赖初始值，容易陷入局部最优解，且由于需要计算协方差矩阵，会更耗时

**3. NDT (Normal Distributions Transform)**
将目标点云划分为一个个网格，计算每个网格的均值和协方差，为每个网格生成概率密度函数，配准时，将源点云投影到网格中，计算落在该网格上的概率得分，寻找一个最优的R，t使得概率总体的概率得分（概率似然）最高
* 优点： 速度快，不需要kd-tree进行最近邻搜索，鲁棒性高，即使初值较差，也能拉回来，比较适合户外大规模点云
* 缺点：极其依赖网格尺寸大小设计，过大或过小都会影响配准精度，极限配准精度不如icp或gicp

### 2.4 影响点云配准精度的因素有什么？
1. 点云本身的质量，如是否有测距噪点或幽灵点，以及点云在空间分布的均匀性
2. 外部环境，高动态的外部环境会严重影响配准，具有高度重复结构的场景也会造成配准失效，长走廊、隧道等几何退化场景 
3. 得到的位姿初始值，偏差严重的初始值会影响精度，以及算法参数设置

### 2.5 如何剔除动态障碍物对点云配准的负面影响？你熟悉哪些离群点滤除算法？
1. 基于深度学习的语义分割方法，对点云做预处理，剔除标签为车辆、行人的点；基于可见性和射线追踪，如果某个在历史子图中为占据的网格在后续发现杯激光穿过，则判定为动态物体并剔除。
2. 离群点剔除
* 统计滤波：对每个点计算附近k个点的平均距离，假设点的分布符合高斯分布，如果平均距离大于$ \mu + \alpha \cdot \sigma $（均值加上$\alpha$倍的标准差），则认定为离群点。
* 半径滤波，设定一个半径R，统计各点半径R内的的邻居数量，少于某个阈值N就认定为是离群点

### 2.6 熟悉哪些回环检测算法？影响回环检测精度、鲁棒性的因素你认为有哪些？
1. Scan Context，将点云投影到极坐标的BEV下，把空间划分为扇形网格，记录网格内点云的最大高度，生成一张2d图像矩阵。
2. 词袋模型：提取图像的 ORB/FAST 局部特征，将它们归类到预先训练好的“视觉字典”中。把一张图片表示为“词汇出现的频率直方图”

### 2.7 你怎么看待scan2scan、scan2map和map2map的配准方式？你倾向使用哪种？
1. scan2scan，使用当前激光帧和上一帧进行匹配，逐渐累加位姿，但是由于每次匹配都有微小的误差，随着时间累计，误差会越来越大
2. scan2map，将过去N帧累积为一个submap，匹配时使用当前帧和子地图进行匹配，由于当前帧是和过去一段时间的地图对齐，能够有效抑制漂移。缺点是需要一直维护这个submap
3. map2map，用过去一段时间的累积map和和历史的map进行匹配，能够有效配准，但是实时性差
