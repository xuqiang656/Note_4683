# 1.ct-lio 编译安装

* 1.1 livox_ros_driver, Livox_sdk 安装

```bash
# Livox_SDK 安装
git clone https://github.com/Livox-SDK/Livox-SDK.git
cd Livox-SDK
cd build && cmake ..
make
sudo make install

#livox_ros_driver 安装（直接安装在ct—ws下，等会直接用于ct-lio的编译）
git clone https://github.com/Livox-SDK/livox_ros_driver.git ct_ws/src
cd ct_ws
catkin_make
```

* 1.2 CT-LIO 编译安装

```bash
# 依赖安装
sudo apt install ros-noetic-jsk-rviz-plugins
sudo apt install libceres-dev
sudo apt install libgoogle-glog-dev
cd ct_ws/src
git clone https://github.com/chengwei0427/ct-lio.git

# 之后需要在thirdparty目录下放置一个g2o-20201223
cd thirdparty/
git clone https://github.com/psg-AI-Tech/g2o-20201223_git.git
cd g2o-20201223/
mkdir build 
cd build/
cmake ..
make
sudo make install  



```
